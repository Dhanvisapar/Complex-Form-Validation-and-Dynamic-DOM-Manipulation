# Complex-Form-Validation-and-Dynamic-DOM-Manipulation

## Objective
To implement advanced form validation, dynamic DOM updates, and client-side routing using JavaScript.

## Features
- Password strength validation
- Real-time DOM updates
- Client-side routing without page reload
- Dynamic content rendering

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open folder in VS Code
2. Open index.html
3. Run with Live Server or open in browser

## Learning Outcomes
- Advanced form validation
- Dynamic DOM manipulation
- Client-side routing
